# GitHub 上傳檔案 — 詳細步驟

你已經建立好一個**空的**倉庫（例如 `intro-site`），現在要上傳三個檔案。

---

## 步驟 1：打開上傳頁面

在倉庫首頁你會看到一段說明，其中有一行藍色連結：

- **「uploading an existing file」**（上傳既有檔案）

**點這行連結**，就會進入上傳檔案的畫面。

（如果沒看到，就點畫面上方的 **Add file**，再選 **Upload files**。）

---

## 步驟 2：拖曳檔案到網頁

1. 打開檔案總管，到這個資料夾：
   ```
   c:\Users\88691\OneDrive\桌面\intro-site
   ```
2. 在資料夾裡**同時選取**這三個檔案：
   - `index.html`
   - `styles.css`
   - `script.js`
   - （按住 **Ctrl** 再依序點三個檔案，就會一起選取）
3. **用滑鼠拖曳**這三個檔案，拖到 GitHub 網頁中間的虛線框裡（上面寫 "Drag files here" 或「將檔案拖到這裡」）。
4. 放開滑鼠，網頁會顯示這三個檔案的檔名，表示已準備上傳。

---

## 步驟 3：寫說明並送出

1. 往下捲動，找到 **Commit changes** 區塊。
2. 在 **Commit message** 的框框裡輸入：`第一次上傳網站`（或任何簡短說明）。
3. 按綠色的 **Commit changes** 按鈕。

---

## 完成

上傳成功後，你會在倉庫裡看到 `index.html`、`styles.css`、`script.js` 三個檔案。

接下來到 **Settings → Pages** 照 DEPLOY.md 第四步設定，就能得到網址。
